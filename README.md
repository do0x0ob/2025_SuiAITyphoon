### Prerequisites
- Sui Wallet

### Installation

1. Clone the repository
```bash
git clone https://github.com/yourusername/memento-os.git
cd memento-os
```
  
2. Install dependencies
```bash
npm install
```
  
3. Set up environment variables
```bash
cp .env.example .env.local
```
  
4. Start the development server
```bash
npm run dev
```
